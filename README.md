## 🚀 Overview
The **Smart Study Planner** is a high-performance web application designed to help engineering students optimize their learning through data-driven scheduling and psychological focus techniques. Built with a **Neo-Minimalist UI**, it applies Design Thinking principles to bridge the gap between academic pressure and productivity.

---

## 💎 Core Features
* **📊 Analytics Hub:** Real-time subject mastery rings and weekly load factor charts for progress analysis.
* **📅 Strategic Planner:** A full 7-day interactive grid to organize academic roadmaps and laboratory sessions.
* **⏱️ Focus Engine:** A professional reverse-countdown timer with built-in motivational "toasts" at custom intervals.
* **🎯 Goal Matrix:** Subject-wise hour targeting with interactive progress sliders to track syllabus coverage.
* **🧠 Neural Advice (Mood Tracker):** An empathy-driven feature providing specific study strategies based on the user's emotional state.
* **🛡️ Academic Guard:** A validated onboarding gate that captures Year, Semester, and Course details to personalize the experience.
